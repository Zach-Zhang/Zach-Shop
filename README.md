 这是一个电商的项目,是一个B2C的电子网站,整个网站采用分布式架构设计,其中构成整个网站的各个子系统包括了后台管理系统,前台系统,订单系统,单点登录系统,搜索系统,购物车系统等;
 开发环境: 
    系统 cenOS6.5
    数据库: myql/mycat,redis
    应用服务器:tomcat
    开发工具: IDEA;Maven Git Navicat;

 使用的技术:
    Dubbo,FastDFS 
    Spring,SpringMVC,Mybatis,
    Lucene,Solr
    Mysql,Redis,Nginx,Quartz ActiveMQ
